# 经验加成器

AkariLevel 提供了经验加成器系统，你可以通过创建自定义加成器给予玩家**永久**/**限时**额外经验加成。

同时，也支持设置加成器的**加成倍率**/**等级组限制**/**经验来源限制**等参数。

## 快速开始

### 1. 创建经验加成器

你可以使用 **/akarilevel booster add 成员名称 经验加成器名称 加成倍率** 命令为玩家新增经验加成器，例如：

```
/akarilevel booster add PlayerName 1.5倍经验加成器 1.5
/akarilevel booster add PlayerName 2.0倍经验加成器 2.0
```

该命令将会为指定成员新增以下参数的经验加成器：

- `ID` 为**随机八位字母数字组合**。
- `名称` 为**指定经验加成器名称**。
- `类型` 为**默认经验加成器类型**。
- `倍率` 为**指定经验加成器倍率**。
- `当前状态` 为**已启用**。
- `起始时间` 为**经验加成器创建时间**。
- `持续时长` 为**永久**。
- `等级组限制` 为**任意等级组**。
- `来源限制` 为 **COMMAND_ADD_EXP**。（通过 **exp add** 命令增加经验）

### 2. 查看经验加成器信息

成员的所有加成器会在**加成器列表**中显示，你可以使用 **/akarilevel booster list 成员名称** 命令查看。

点击加成器右侧的 **<ℹ>** 按钮可以查看加成器详细信息。

### 3. 启用/禁用经验加成器

你可以使用 **/akarilevel booster disable 成员名称 经验加成器ID** 命令禁用加成器。

在禁用期间，**加成器不会生效，且不会计算时长**。

如需再次启用加成器，执行 **/akarilevel booster enable 成员名称 经验加成器ID** 命令即可。

## 参数详解

你可以在 **booster add** 命令中使用 `--键=值` 提供命令参数，例如：

```
/akarilevel booster add PlayerName 1.5倍经验加成器 1.5 --id=Test --type=活动经验加成器组 --duration=1d12h --levelGroup=Example --source=MYTHICMOBS_DROP_EXP
```

- `id`: ID，默认为随机八位字母数字组合。
- `type`: 类型，默认为空字符串。**同种类型取最大倍率，不同类型倍率相乘**。
- `duration`: 持续时长，默认为永久。单位 **d天**/**h时**/**m分**/**s秒**。
- `levelGroup`: 等级组限制，默认为空字符串。
- `source`: 来源限制，默认为 **COMMAND_ADD_EXP**。
