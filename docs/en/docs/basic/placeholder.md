# PlaceholderAPI Variables

AkariLevel provides rich **PlaceholderAPI** variables.

Variable expansion will be automatically registered when the plugin is enabled, no manual installation required.

## Variable List

| Variable                                                                       | Description                                         |
|--------------------------------------------------------------------------------|-----------------------------------------------------|
| `%AkariLevel_LevelGroupEditName_Name%`                                         | Level group edit name                               |
| `%AkariLevel_LevelGroupEditName_Display%`                                      | Level group display name                            |
| `%AkariLevel_LevelGroupEditName_Level%`                                        | Current level                                       |
| `%AkariLevel_LevelGroupEditName_LastLevel%`                                    | Previous level                                      |
| `%AkariLevel_LevelGroupEditName_NextLevel%`                                    | Next level                                          |
| `%AkariLevel_LevelGroupEditName_MinLevel%`                                     | Minimum level                                       |
| `%AkariLevel_LevelGroupEditName_MaxLevel%`                                     | Maximum level                                       |
| `%AkariLevel_LevelGroupEditName_Exp%`                                          | Current experience                                  |
| `%AkariLevel_LevelGroupEditName_LevelName%`                                    | Current level name                                  |
| `%AkariLevel_LevelGroupEditName_LastLevelName%`                                | Previous level name                                 |
| `%AkariLevel_LevelGroupEditName_NextLevelName%`                                | Next level name                                     |
| `%AkariLevel_LevelGroupEditName_LevelExp%`                                     | Experience required to level up to current level    |
| `%AkariLevel_LevelGroupEditName_LastLevelExp%`                                 | Experience required to level up to previous level   |
| `%AkariLevel_LevelGroupEditName_NextLevelExp%`                                 | Experience required to level up to next level       |
| `%AkariLevel_LevelGroupEditName_LevelExpFrom_OldLevel%`                        | Experience required from old level to current level |
| `%AkariLevel_LevelGroupEditName_LevelExpTo_NewLevel%`                          | Experience required from current level to new level |
| `%AkariLevel_LevelGroupEditName_LevelExpFromTo_OldLevel_NewLevel%`             | Experience required from old level to new level     |
| `%AkariLevel_LevelGroupEditName_LevelProgressPercent%`                         | Level progress percentage                           |
| `%AkariLevel_LevelGroupEditName_ExpProgressPercent%`                           | Experience progress percentage                      |
| `%AkariLevel_LevelGroupEditName_LevelProgressBar_EmptyChar_FullChar_Length%`   | Level progress bar                                  |
| `%AkariLevel_LevelGroupEditName_ExpProgressPercent_EmptyChar_FullChar_Length%` | Experience progress bar                             |
