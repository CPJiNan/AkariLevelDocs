# JavaScript Scripts

AkariLevel provides support for JavaScript, you can use scripts to register **commands**/**listeners**
/**scheduled tasks**/**placeholders** to implement custom functions.

This tutorial mainly describes the usage of built-in scripts. If you want to learn how to write custom scripts, please
go to **Advanced Tutorial - JavaScript Scripts**.

## Quick Start

Script files are located in the `plugins/AkariLevel/script` path and will be automatically loaded when the plugin is
enabled.

Some scripts may require configuration before use. After modification, reload the plugin to take effect.
